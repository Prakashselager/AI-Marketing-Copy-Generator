# AI Marketing Copy Generator 🚀

[![PHP Version](https://img.shields.io/badge/PHP-8.0+-blue.svg)](https://php.net)
[![MySQL](https://img.shields.io/badge/MySQL-5.7+-orange.svg)](https://mysql.com)

A powerful web application that leverages artificial intelligence to generate professional marketing copy for multiple platforms. Create engaging content for Facebook, Instagram, LinkedIn, Google Ads, and Email in seconds.



## ✨ Features

### For Users
- **AI-Powered Copy Generation**: Generate marketing copy using advanced language models
- **Multi-Platform Support**: Create content optimized for Facebook, Instagram, LinkedIn, Google Ads, and Email
- **Template System**: Use pre-built templates or create custom ones
- **Credit Management**: Track and manage usage credits
- **Save & Export**: Save generated copies and export them in various formats
- **User Dashboard**: View generation history and manage account settings

### For Admins
- **Comprehensive Dashboard**: Real-time analytics and statistics
- **User Management**: Add, edit, delete users and manage credits
- **Template Management**: Create and modify copy templates
- **Settings Control**: Configure API keys, credit defaults, and system settings
- **Usage Monitoring**: Track platform usage and user activity

## 🛠️ Tech Stack

### Backend
- **PHP 8.0+** - Core programming language
- **MySQL 5.7+** - Database management
- **OpenAI API** - AI copy generation
- **PDO** - Database abstraction layer

### Frontend
- **HTML5** - Structure
- **CSS3** - Styling with modern flexbox/grid
- **JavaScript** - Interactive features
- **Font Awesome 6** - Icons
- **Google Fonts (Inter)** - Typography

### Security
- Password hashing (bcrypt)
- Prepared statements (SQL injection prevention)
- XSS protection
- CSRF tokens
- Session management

## 📦 Installation

### Prerequisites
- PHP 8.0 or higher
- MySQL 5.7 or higher
- Web server (Apache/Nginx)
- OpenAI API key
- Composer (optional)
